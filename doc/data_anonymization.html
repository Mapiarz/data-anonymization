compute_stats/3

compute_stats(Directory, Include_duplicates, Group_size) -> list()

  • Directory = string()
  • Include_duplicates = yes | no
  • Group_size = number()

Parses JSON dataset, extracts airline ticket prices, generates tuples, aggregates them, anonymizes and then computes stats: AVG, MED, MIN, MAX. Reads the dataset from Directory path string, Include_duplicates specifies whether to return duplicate tuples (yes) per user or remove them (no). Group_size specifies size of histogram buckets.
